# PYTHON-Quiz-Game-OOP-
A Command-line Interface (CLI) Quiz Game built with python using Object-Oriented Programming(OOP) principles. Features dynamic scoring and question management.

# Project Structure :
```text
.
├── main.py
├── quiz_brain.py
├── question_model.py
├── data.py
└── README.md
```

# FEATURES :
 1. **Dynamic scoring** - Tracks your score as you play.
 2. **OOP Design**      - Utilizes classes for questions and game logic for clean, maintainable code.
 3. **Interactive**     - Real-time feedback on your answers.
